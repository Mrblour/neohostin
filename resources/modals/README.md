# Modals

Esta carpeta está pensada para guardar vistas parciales de modales (por ejemplo, formularios, avisos, diálogos) que luego puedas cargar dinámicamente desde tu SPA.

Ejemplos de futuros archivos:
- `alerta-eliminacion.html`
- `form-contacto-rapido.html`
- `modal-login-light.html`

Puedes usar la misma técnica que con `partials` (fetch + innerHTML) para inyectar estos modales donde los necesites.
