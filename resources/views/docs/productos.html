¿Qué es Mis Productos?

Mis Productos es tu panel centralizado donde podrás ver y gestionar todos los servicios que has contratado con Neohosting. Desde aquí tendrás acceso rápido a todos tus servidores, VPS y servicios activos.

¿Qué puedes hacer?

Una vez que tengas productos activos, podrás:

  • Ver todos tus servicios: Lista completa de servidores, VPS y otros productos contratados
  • Estado en tiempo real: Verificar si tus servicios están activos, suspendidos o en mantenimiento
  • Acceso rápido: Enlaces directos al panel de control de cada servicio
  • Información de facturación: Ver próximas fechas de renovación y precios
  • Gestión de recursos: Monitorear uso de CPU, RAM y almacenamiento
  • Acciones rápidas: Reiniciar, detener o iniciar tus servicios

Cómo adquirir un producto

Paso 1: Explora nuestros servicios

Visita nuestras páginas de productos para conocer las opciones disponibles:

  • Servidores Minecraft - Hosting optimizado para Minecraft
  • Servidores VPS - Servidores virtuales privados con control total
  • Servidores Dedicados - Hardware exclusivo para tu proyecto
  • Game Servers - Hosting para múltiples juegos

Paso 2: Elige tu plan

Cada servicio ofrece diferentes planes según tus necesidades. Compara las características, recursos y precios para encontrar el plan perfecto para ti.

Paso 3: Crea una cuenta

Si aún no tienes una cuenta, necesitarás crear una. Solo necesitas proporcionar:

  • Nombre completo
  • Dirección de correo electrónico
  • Contraseña segura

Paso 4: Configura tu servicio

Personaliza tu servicio seleccionando:

  • Ubicación del servidor (Estados Unidos, Europa, etc.)
  • Sistema operativo (para VPS y dedicados)
  • Versión del juego (para servidores de juegos)
  • Recursos adicionales si los necesitas

Paso 5: Completa el pago

Procesamos pagos de forma segura mediante:

  • Tarjetas de crédito/débito
  • PayPal
  • Transferencias bancarias
  • Criptomonedas (Bitcoin, Ethereum)

Paso 6: Activación instantánea

Una vez confirmado el pago, tu servicio se activa automáticamente en menos de 5 minutos. Recibirás un correo con:

  • Datos de acceso al panel de control
  • Información de conexión (IP, puerto, credenciales)
  • Guía de inicio rápido
  • Enlaces a documentación y soporte

Información básica de tus productos

Para cada producto activo, podrás ver:

Información General

  • Nombre del servicio: Identificador único de tu producto
  • Tipo de servicio: Minecraft, VPS, Dedicado, etc.
  • Plan contratado: Nombre y características del plan
  • Estado: Activo, Suspendido, En mantenimiento
  • Ubicación: Centro de datos donde está alojado

Recursos Asignados

  • CPU: Núcleos y frecuencia asignados
  • RAM: Memoria disponible
  • Almacenamiento: Espacio en disco (SSD/NVMe)
  • Ancho de banda: Transferencia mensual incluida
  • IP: Dirección IP asignada

Información de Facturación

  • Precio: Costo mensual/anual del servicio
  • Próxima renovación: Fecha del próximo cobro
  • Método de pago: Forma de pago configurada
  • Historial: Facturas anteriores

Gestión de productos

Acciones disponibles para cada producto:

  • Acceder al panel: Gestiona tu servidor desde el panel de control
  • Reiniciar: Reinicia tu servicio cuando lo necesites
  • Actualizar plan: Cambia a un plan superior o inferior
  • Renovar: Extiende la duración de tu servicio
  • Cancelar: Cancela el servicio (disponible hasta 24h antes de la renovación)
  • Soporte: Abre un ticket de soporte para ese servicio específico
