Cómo Configurar tu Servidor FiveM

Guía completa para crear tu servidor de GTA V Roleplay

Requiere licencia de FiveM (Patreon)

Introducción

FiveM es una plataforma de modificación para GTA V que permite crear servidores multijugador personalizados. Este tutorial te guiará en la configuración de tu servidor.

1

Obtén tu Licencia de FiveM

Necesitas una licencia de servidor de FiveM para ejecutar tu servidor.

1

Visita keymaster.fivem.net

2

Inicia sesión con tu cuenta de Cfx.re

3

Genera una nueva clave de servidor

Importante: Guarda tu clave de servidor en un lugar seguro. La necesitarás en el paso 3.
2

Accede al Panel de Control

Inicia sesión en el panel de control de Neohosting.

URL del Panel
https://panel.neohosting.com
3

Configura server.cfg

Edita el archivo de configuración principal de tu servidor.

Configuración básica:
sv_licenseKey "tu-clave-de-servidor"
sv_hostname "Mi Servidor FiveM"
sv_maxclients 32
sv_endpointprivacy true
4

Instala un Framework

Elige e instala un framework para tu servidor roleplay.

ESX Legacy
Más popular

Framework completo para roleplay con economía, trabajos y más

QBCore
Moderno

Framework optimizado y actualizado regularmente

vRP
Clásico

Framework veterano con amplia comunidad

5

Instala Recursos/Scripts

Añade scripts y recursos para personalizar tu servidor.

Pasos para instalar recursos:
1

Descarga el recurso desde GitHub o foros de FiveM

2

Sube la carpeta a /resources/

3

Añade ensure nombre-recurso en server.cfg

4

Reinicia el servidor

6

Configura la Base de Datos

La mayoría de frameworks requieren una base de datos MySQL.

Conexión MySQL en server.cfg:
set mysql_connection_string "mysql://usuario:contraseña@localhost/nombre_bd"
Los datos de tu base de datos MySQL se encuentran en el panel de control de Neohosting.
7

Inicia el Servidor

Arranca tu servidor y conéctate desde FiveM.

Servidor iniciándose...
El servidor puede tardar 1-2 minutos en cargar todos los recursos.
Conectarse desde FiveM
Presiona F8 en FiveM y escribe:
connect tu-ip:30120

Recursos Recomendados

oxmysql Base de datos

Conector MySQL optimizado para FiveM

pma-voice Voz

Sistema de voz 3D para roleplay inmersivo

progressBars UI

Barras de progreso para acciones del jugador
