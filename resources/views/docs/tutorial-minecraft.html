Cómo Configurar tu Servidor Minecraft

Guía paso a paso para poner en marcha tu servidor

Tutorial actualizado para Minecraft 1.20+

Introducción

Una vez que hayas contratado tu servidor Minecraft con Neohosting, recibirás un correo con los datos de acceso. Este tutorial te guiará a través de los pasos necesarios para configurar y personalizar tu servidor.

1

Accede al Panel de Control

Inicia sesión en tu panel de control usando las credenciales que recibiste por email.

URL del Panel
https://panel.neohosting.com

Tip: Guarda tus credenciales en un lugar seguro. Puedes usar un gestor de contraseñas.

2

Selecciona la Versión de Minecraft

Desde el panel, elige la versión de Minecraft que deseas usar.

Vanilla
Minecraft oficial sin modificaciones
Spigot / Paper
Para plugins y mejor rendimiento
Forge / Fabric
Para mods
3

Configura las Propiedades del Servidor

Personaliza la configuración básica de tu servidor editando el archivo server.properties

Configuraciones importantes:
server-port 25565
max-players 20
difficulty normal
gamemode survival
pvp true
4

Inicia el Servidor

Una vez configurado, inicia tu servidor desde el panel de control.

Servidor iniciándose...
El servidor tardará aproximadamente 30-60 segundos en estar completamente operativo.
5

Conéctate al Servidor

Abre Minecraft y conéctate usando la IP de tu servidor.

Dirección del Servidor
tu-servidor.neohosting.com:25565
Nota: La IP exacta de tu servidor se encuentra en el panel de control y en el email de bienvenida.
6

Hazte Operador (OP)

Para tener permisos de administrador en el servidor, necesitas hacerte OP.

Comando en la Consola
op TuNombreDeUsuario
Ejecuta este comando desde la consola del panel de control, reemplazando "TuNombreDeUsuario" con tu nombre de Minecraft.
7

Instala Plugins (Opcional)

Si usas Spigot/Paper, puedes instalar plugins para añadir funcionalidades.

Pasos para instalar plugins:
1

Descarga el plugin (.jar) desde sitios como SpigotMC o Bukkit

2

Sube el archivo a la carpeta /plugins/

3

Reinicia el servidor desde el panel

Plugins recomendados: EssentialsX, WorldEdit, LuckPerms, Vault

Comandos Útiles

/whitelist add [jugador] Añadir a whitelist

Permite que un jugador específico se conecte al servidor

/ban [jugador] Banear jugador

Expulsa permanentemente a un jugador del servidor

/gamemode [modo] [jugador] Cambiar modo de juego

Cambia entre survival, creative, adventure o spectator

/tp [jugador] [destino] Teletransportar

Teletransporta un jugador a otro jugador o coordenadas
