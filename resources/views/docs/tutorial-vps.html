Cómo Configurar tu Servidor VPS

Guía completa para configurar y asegurar tu VPS desde cero

Tutorial para Ubuntu 22.04 LTS y Debian 12

Introducción

Este tutorial te guiará a través de la configuración inicial de tu VPS, desde la primera conexión hasta tener un servidor seguro y listo para usar.

1

Conecta por SSH

Usa SSH para conectarte a tu VPS por primera vez.

Comando SSH
ssh root@tu-ip-del-vps
Windows: Usa PuTTY o Windows Terminal
Mac/Linux: Usa la terminal integrada
2

Actualiza el Sistema

Actualiza todos los paquetes a sus últimas versiones.

Para Ubuntu/Debian:
sudo apt update && sudo apt upgrade -y
3

Crea un Nuevo Usuario

Por seguridad, crea un usuario no-root para uso diario.

Comandos:
adduser tunombre
usermod -aG sudo tunombre
Importante: Reemplaza "tunombre" con el nombre de usuario que desees.
4

Configura el Firewall

Instala y configura UFW (Uncomplicated Firewall) para proteger tu servidor.

Comandos de configuración:
sudo apt install ufw
sudo ufw allow 22/tcp // SSH
sudo ufw allow 80/tcp // HTTP
sudo ufw allow 443/tcp // HTTPS
sudo ufw enable
5

Instala Software Básico

Instala herramientas esenciales para gestionar tu servidor.

Paquetes recomendados:
sudo apt install -y curl wget git htop nano vim
6

Instala un Servidor Web (Opcional)

Si planeas alojar sitios web, instala Nginx o Apache.

Opción 1: Nginx (Recomendado)
sudo apt install nginx
Opción 2: Apache
sudo apt install apache2
7

Configura SSH con Claves

Mejora la seguridad usando autenticación por clave SSH en lugar de contraseña.

En tu computadora local:
Generar clave SSH:
ssh-keygen -t rsa -b 4096
Copiar clave al servidor:
ssh-copy-id tunombre@tu-ip-del-vps

Comandos Útiles para VPS

htop Monitor de recursos

Ver uso de CPU, RAM y procesos en tiempo real

df -h Espacio en disco

Ver espacio disponible en el disco

sudo systemctl status [servicio] Estado de servicio

Ver el estado de un servicio (nginx, apache, etc.)

sudo reboot Reiniciar servidor

Reinicia el VPS (úsalo con precaución)
